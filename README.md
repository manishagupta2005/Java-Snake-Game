Java Snake Game

This is a classic Snake Game built using Java and Swing.
I created this project to practice object-oriented programming and understand how game logic works in Java.

Features

* Real-time keyboard controls
* Snake movement and growth
* Collision detection (walls & self)
* Score tracking

 Tech Used

* Java
* Java Swing (for GUI)

How to Run

1. Clone the repository
2. Open in any Java IDE (IntelliJ / Eclipse / NetBeans)
3. Run the main class


What I Learned

* Handling keyboard events
* Game loop logic
* Collision detection
* GUI development using Swing
